# Bright-Coffee-Shop-Sales-Analysis-
This case study explores the Bright Coffee Shop sales dataset, which contains daily transactional data from the business. This analysis aims to transform raw data into meaningful business insights to drive revenue growth and improve product performance.

The objective is to leverage data analytics, SQL, and visualization techniques to uncover key patterns in sales performance. This includes identifying top-performing products, understanding peak business hours, and analyzing sales trends across different time intervals.

Through structured data processing and analysis using tools such as Databricks, this project will deliver actionable recommendations to support strategic decision-making and enhance overall business performance.
